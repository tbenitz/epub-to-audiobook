# EPUB → Audiobook (in-browser)

Upload any EPUB and listen to it with Supertonic — the fastest on-device TTS in the world.

- 100% client-side  
- No data sent anywhere  
- Works offline after first load  
- Powered by [Supertonic](https://github.com/supertone-inc/supertonic) + EPUB.js

Just open index.html or deploy to GitHub Pages!
